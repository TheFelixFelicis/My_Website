# Alexander Vatamidis Norrstam - Portfolio

A minimalist, aesthetic, portfolio website built entirely from scratch to showcase my work in DS.

**Live Site:** [alexandernorrstam.com](https://alexandernorrstam.com)

## Project Overview

This website relies on semantic HTML5 and a custom CSS3 design system.

## Tech Stack

* **Core:** Semantic HTML5
* **Styling:** Vanilla CSS3 (Variables/Custom Properties)
* **Fonts:** Playfair Display (Headers) & Lora (Body)
* **Hosting:** Cloudflare

## Project Structure

* `index.html` - Landing page
* `projects.html` - Portfolio grid
* `style.css` - Global design system
* `about.html` - Biographical information
* `education.html` - Academic background

## How to Run

Since this is a static site, no build step is required.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/TheFelixFelicis/My_Website.git](https://github.com/TheFelixFelicis/My_Website.git)
    ```
2.  **Open locally:**
    Open `index.html` in any web browser.
