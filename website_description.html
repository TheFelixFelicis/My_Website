Building This Website

Project Overview

Designed and developed entirely from scratch, this website serves as a central hub for my work as a whole. Avoiding heavy web frameworks, the site relies on HTML5 and vanilla CSS3 for a lightweight final product. The architecture balances a distinct personal aesthetic with interaction design principles. The website utilizes CSS variables to enforce strict consistency across color palettes, spacing, and breakpoints.

The challenge was creating a visual rhythm without using images or heavy graphics. Instead, the site uses whitespace, borders, and precise typography as the primary visual elements. The intention of this approach was to allow the content to be the focal point of attention, regardless of whether the subject is a Machine Learning model or an inquiry into the ethics of AI.
